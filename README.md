## Overview

This project is a Star Wars data browsing app in React using SWAPI API. It allows you to browse the list of characters, planets, films, starships, species and vehicles in the provided SWAPI database. <br>

## Packages

- [React Router Dom](https://reacttraining.com/react-router/web/guides/quick-start)
- [Axios](https://github.com/axios/axios)

## Installation and usage

Be sure, you have installed all dependencies and applications to run React project on your computer : [Getting Started with React](https://reactjs.org/docs/getting-started.html).

### Running the project

Clone this repository :

```
git clone https://github.com/cadupuy/swapi-front
cd swapi-front
```

Install packages :

```
yarn
```

When installation is complete, run it :

```
yarn start
```
